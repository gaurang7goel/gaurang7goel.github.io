## 🚀 Quick Start Guide - Portfolio Setup

### Step 1: Customize Your Information

1. **Open `index.html`** and search for these placeholders:
   - Replace `gaurang.goel@example.com` with your actual email
   - Update LinkedIn URL with your profile
   - Update GitHub URL with your profile
   - Add your Twitter handle (if any)

2. **Update certifications** (if different):

   ```html
   <h3>DP-203</h3>
   <p class="cert-description">Data Engineering on Microsoft Azure</p>
   ```

3. **Customize the skills section** with your specific technologies

### Step 2: Personalize Colors (Optional)

Edit `styles.css` and modify the color variables:

```css
:root {
  --primary: #00d4ff; /* Change this to your favorite color */
  --primary-dark: #0099cc;
  --primary-light: #4df5ff;
  --secondary: #ff006e; /* Accent color */
  --accent: #8338ec; /* Another accent */
}
```

### Step 3: Test Locally

1. Open `index.html` directly in your browser
2. Test all navigation links
3. Test the contact form
4. Check responsive design (use DevTools for mobile view)

### Step 4: Deploy to GitHub Pages

#### Option A: Using Git Command Line

```bash
cd d:\Downloads\gaurang7goel.github.io
git add .
git commit -m "Update portfolio website with latest experience and skills"
git push origin main
```

#### Option B: Using GitHub Desktop

1. Open GitHub Desktop
2. Select your repository
3. Click "Commit to main"
4. Click "Push origin"

### Step 5: Verify Deployment

1. Go to: `https://gaurang7goel.github.io`
2. Wait 2-3 minutes for GitHub Pages to build
3. If not showing, check Repository Settings → Pages
4. Ensure "Deploy from a branch" is selected with "main" branch
