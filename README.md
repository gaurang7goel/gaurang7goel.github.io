# Gaurang Goel Portfolio

Static GitHub Pages portfolio for Gaurang Goel, focused on Azure data engineering,
governed data platforms, medallion architecture, observability, automation, and
Microsoft Fabric learning.

## Site Sections

- Hero with animated data fabric canvas
- Hiring-manager profile snapshot
- Data engineering value areas
- Azure, Databricks, monitoring, governance, and platform stack
- Microsoft certifications: DP-900, DP-203, DP-700
- Automation, AI, and Fabric learning vision
- Contact links and mailto form

## Files

```text
index.html
styles.css
script.js
```

## Local Preview

Open `index.html` directly in a browser. No build step is required.

## Deployment

Push the repository to GitHub and enable GitHub Pages from the repository root.
The site is intended to publish at:

```text
https://gaurang7goel.github.io
```
